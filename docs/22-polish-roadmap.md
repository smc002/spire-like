# 22 从 Demo 到正式游戏的表现层路线图

本文档专注**表现层**（visual / feel / juice）与正式游戏的差距。代码功能层（更多卡、更多敌人、地图、商店、Boss 等）请看 [20-dev-roadmap.md](20-dev-roadmap.md)。

## 当前状态 vs 目标

| 维度 | 当前 demo | 正式游戏目标 |
|------|----------|-------------|
| 卡牌视觉 | 纯色矩形 + 文字 | 卡牌底框美术、稀有度边框、插图 |
| 卡牌交互 | 拖拽（瞬间响应） | 悬停放大、拖拽跟随、伤害预测、释放动画 |
| 敌人视觉 | 文字 + 占位框 | sprite、idle 动画、受击动画 |
| 信息呈现 | 全是文字标签 | HP 条、能量球、状态图标 + tooltip |
| 战斗反馈 | 立即结算无动画 | 飘字、闪烁、屏幕震动、特效 |
| 时序 | 同步瞬间 | 敌人回合分步、关键时刻定格 |
| 音频 | 无 | BGM、SFX、UI 音效 |
| 流程 | 一场战斗 + Restart | 主菜单、暂停、设置、卡组查看、跑团结束页 |

## 类别分解（按优先级排序）

### P0 — 核心手感（直接影响"想不想玩")

**1. 卡牌 hover 反馈**
- 鼠标进入卡牌 → 缓动放大 1.08x + 上移 20px
- 鼠标离开 → 缓动还原
- 实现：`CardView.mouse_entered/exited` + Tween

**2. 伤害预测**
- 拖动攻击卡到敌人身上时，敌人面板显示"-X HP"预测值
- 预测计算包含：力量 + 玩家虚弱 + 目标易伤 + multi-hits
- AOE 卡显示对所有敌人的预测
- 实现：BattleScene `_process` 轮询鼠标位置 + 计算函数

**3. 拖拽落点提示**
- 拖动有目标卡：敌人面板高亮 / 描边
- 拖动无目标卡：手牌上方"落点区"显示淡色提示条
- 拖到无效区域：卡边框变红警示

**4. 卡牌出牌动画**
- 卡向目标飞出 + 缩小 + 淡出（0.25s）
- 自我/全体卡向上飘 + 淡出
- 当前是瞬间消失，损失感很重

**5. 受击反馈**
- 敌人/玩家受伤时 sprite 红色闪烁 + 轻微抖动 0.15s
- 实现：modulate Tween，position Tween 轻微偏移

---

### P1 — 信息清晰度（影响"看不看得懂")

**6. HP 条**
- 不只是 "HP 65/80" 文字，要 ProgressBar
- 颜色：>50% 绿，25-50% 黄，<25% 红
- 受伤时延迟显示损失（白色 trail 渐隐）

**7. 能量球**
- 左下圆形显示当前/上限（例如 `3/3`）
- 实现：圆形 Panel + 中心 Label

**8. 状态图标**
- 当前是 "vulnerable:2" 文字
- 改为：图标 + 层数小角标
- hover 显示 tooltip：名称 + 描述 + 衰减规则
- 实现：StatusChip 节点（TextureRect + Label）

**9. 意图图标**
- 当前是 "Atk 6" 文字
- 改为：拳头/盾牌/箭头图标 + 数字
- 多段攻击显示 "x N"
- 意图 hover 显示完整说明

**10. 飘字（Damage Numbers）**
- 伤害、护甲、回血、状态层数都飘字
- 红色伤害、蓝色护甲、绿色回血、紫色 buff、橙色 debuff
- 飘字向上漂 0.6s + 淡出
- 实现：单独 Label 节点 + Tween，挂在专门的浮层

**11. 牌堆点击查看**
- 点 Draw Pile → 弹出当前抽牌堆里的所有卡（随机顺序，玩家看不到顺序）
- 点 Discard Pile → 弃牌堆（按时间顺序）
- 点 Exhaust Pile → 消耗堆
- 实现：弹窗 + CardView 网格布局

---

### P2 — 节奏感（影响"打起来爽不爽")

**12. 敌人回合分步执行**
- 当前是 `end_player_turn` 同步连续触发所有敌人意图，瞬间结算
- 改为：每个敌人动作之间 0.4s 间隔，每段攻击之间 0.2s
- 实现：把 `_enemy_turn` 改成 async / await 链
- 关键：玩家要看清"谁打了我多少"

**13. 抽牌/弃牌动画**
- 抽牌：从抽牌堆位置以弧线飞到手牌槽（0.2s/张）
- 弃牌：从手牌位置滑到弃牌堆（0.15s/张）
- 同时抽多张时错开 0.05s
- 实现：CardView 出生在抽牌堆 → tween 到目标位置

**14. 消耗动画**
- 卡牌淡出 + 缩小 + 朝消耗堆飞 + 红/紫色 modulate
- 对 Power 卡、exhaust 卡牌都要

**15. 战斗结束演出**
- 胜利：1s 全屏轻微变暗 + 敌人淡出
- 失败：屏幕红色闪烁 + 减速 + "Defeated" 慢慢显现
- 当前是瞬间显示遮罩

**16. 卡牌升级演出**
- 篝火节点选卡升级时金光特效（暂未实现，等阶段 8）

---

### P3 — 沉浸感（影响"愿不愿意一直玩")

**17. 音频**
- BGM：战斗 / 地图 / 商店 / Boss / 主菜单（各一首）
- SFX：
  - 攻击卡播放（按伤害大小）
  - 技能卡播放（防御音）
  - 能力卡播放（buff 音）
  - 受击音
  - 抽牌音
  - 弃牌洗回音
  - UI hover / click / 错误音
- 来源：免费资源站（OpenGameArt、Freesound、Pixabay）或委托

**18. 屏幕震动**
- 大伤害（>=15）触发屏幕震动 0.15s 幅度 4px
- Boss 出场震动
- 实现：Camera2D + offset 抖动 Tween，或 ColorRect overlay shake

**19. 粒子效果**
- 攻击卡：火星/红色碎片
- 防御卡：蓝色光环
- 状态施加：对应颜色光环
- 死亡：烟雾/碎片
- 实现：GPUParticles2D

**20. 背景**
- 当前是纯色背景
- 改为：场景背景（按层数/敌人类型变化）
- 视差滚动（多层背景以不同速度移动）

---

### P4 — 商业级完成度（影响"是不是产品")

**21. 美术资源全替换**
- 卡牌底框（按 ATTACK/SKILL/POWER 三套）
- 稀有度边框（普通/不普通/稀有视觉差异）
- 卡牌插图（每张卡一张图）
- 敌人 sprite + 帧动画
- UI 元素：按钮、面板、图标、字体

**22. 主菜单**
- 新跑团 / 继续（如有存档）/ 卡牌图鉴 / 设置 / 退出
- 背景动效

**23. 暂停菜单**
- Esc 暂停 / 继续 / 放弃跑团 / 退出主菜单 / 设置

**24. 设置菜单**
- 音量（主/BGM/SFX 分轨）
- 分辨率 / 窗口模式
- 按键自定义
- 语言（中英文）

**25. 卡组查看（战斗中可调）**
- 任何时候按 Tab → 浮层显示当前牌组（区分 draw/discard/exhaust）

**26. 跑团结束统计**
- 用时、击杀数、总伤害、最高单回合伤害
- 最终牌组、获得的遗物
- "再来一局" / "回主菜单"

**27. 教学 / 引导**
- 第一次跑团：箭头提示"拖卡到敌人身上"、"点击 End Turn"
- 关键概念解释：易伤、虚弱、消耗、能力

**28. 存档 / 续跑**
- 自动存档每个节点
- 退出可恢复

## 建议执行顺序

如果以"最少投入获得最大玩家感知提升"为目标：

```
本次任务（任务 2 范围内）:
  P0-1 卡牌 hover           ← 立刻能看到
  P0-2 伤害预测             ← 用户明确要求
  P0-3 拖拽落点提示          ← 配套体验
  P1-10 飘字基础版          ← 让战斗"活"起来

下一批（任务 2.5）:
  P0-4 卡牌出牌动画
  P0-5 受击反馈
  P1-6 HP 条
  P1-7 能量球

再一批（任务 2.6）:
  P2-12 敌人回合分步
  P2-13 抽牌/弃牌动画
  P1-8 状态图标 + tooltip

之后（任务 1 美术替换）:
  P4-21 资源全替换
  P1-9 意图图标

最后（任务 3+ 流程完整化）:
  P4-22..28
  P3 音频
```

每批做完都应该能感觉到"游戏向正式形态又靠近一步"。

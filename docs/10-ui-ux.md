## 输入模型

### 鼠标主导

| 操作 | 触发 |
|------|------|
| 出牌（无目标） | 卡牌拖到角色身上以外 / 双击 |
| 出牌（指敌） | 卡牌拖到敌人身上 |
| 取消出牌 | 拖回手牌位置 |
| 查看卡牌大图 | 鼠标悬停 |
| 查看意图详情 | 鼠标悬停敌人 |
| 查看遗物详情 | 鼠标悬停遗物 |
| 查看卡组 | 点击「卡组」按钮 |
| 结束回合 | 点击「结束回合」按钮 |

### 键盘快捷键

| 键 | 操作 |
|---|------|
| 1-9 | 出第 N 张手牌 |
| Space / E | 结束回合 |
| Tab | 显示卡组 |
| Esc | 暂停 / 关闭弹窗 |

## 卡牌视觉

卡牌大小：约 200×280 px。组件：

```
┌────────────────────┐
│ [费用]      [类型] │
│ ┌──────────────┐   │
│ │  [插图区]    │   │
│ └──────────────┘   │
│  卡牌名（黑底白字）│
│ ─────────────────  │
│  效果描述文本      │
│                    │
└────────────────────┘
```

- 颜色按职业：战士红/橙
- 升级版本卡名前加 `+`、底色稍深
- 不可出（能量不足/限制）时半透明 + 红色边框

## 状态可视化

- 每个状态在角色头像下方一行图标 + 层数
- 状态变化时图标弹跳 + 数字飘字
- 力量/敏捷等永久 buff 图标在左侧，临时回合 debuff 在右侧

## 动画与节奏

| 行为 | 时长 |
|------|------|
| 抽 1 张卡 | 0.15s |
| 出牌结算 | 0.3-0.5s |
| 敌人意图执行（每段） | 0.4s |
| 伤害飘字 | 0.6s |
| 战斗胜利结算 | 1.0s |

总原则：玩家每个回合不应超过 5-8s 等待，敌方回合不应超过 3s 等待。所有动画可被玩家点击跳过（除了关键演出）。

## 颜色语义

| 颜色 | 含义 |
|------|------|
| 红 | 攻击、HP、负面 |
| 蓝 | 护甲、技能 |
| 紫 | 能力 |
| 灰/白 | 基础卡 |
| 金 | 稀有、金币 |

## 字体

- 标题：粗衬线（如 Cinzel） 
- 正文：无衬线（如 Noto Sans）
- 数字（HP/伤害/能量）：高辨识等宽字体

中文需要至少 1 套支持简体的字体（推荐思源黑体/Source Han Sans）。

## 屏幕分辨率

- 设计分辨率：1280×720
- 拉伸模式：`canvas_items` + `keep`（保持纵横比，添加黑边）
- UI 尽量用 anchor + container 适配

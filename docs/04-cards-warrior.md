# 04 战士卡牌

## 战士的职业身份

战士是高血量近战职业。核心标签：

- **力量缩放**：堆叠力量 buff 让单卡攻击力线性放大
- **护甲交换**：用 HP 换取强力效果（自损卡），用攻击换护甲（暴怒/荆棘）
- **消耗机制**：部分高强度卡牌出一次就消耗，强迫一次性高爆发决策
- **生命管理**：起手 HP 80（最高职业），最大值随跑团缓慢提升

### 起手数据

| 属性 | 数值 |
|------|------|
| 最大 HP | 80 |

| 稀有度 | 标识 | 出现频率（战斗奖励） | 备注 |

| 基础（Basic） | 灰 | 仅起始卡组 | 无奖励池 |

## 升级（+）规则

每张卡有未升级和升级两个版本。升级一般做以下中的一项：

| 升级模式 | 示例 |
|---------|------|
| 数值提升 | 攻击伤害 +3、格挡 +3、状态层数 +1 |
| 费用降低 | 2 费 → 1 费 |
| 增加附带效果 | 攻击 → 攻击 + 抽 1 张 |
| 改变机制 | 「弃 1 抽 1」→「弃 1 抽 2」 |
| 移除副作用 | 「+消耗」→ 「无消耗」 |

升级一般在篝火节点选 1 张卡升级；事件、某些遗物也可触发。

## 构筑流派（设计验证）

战士起始牌池要至少支持 3 种流派各自能通关 Act 1：

1. **力量流**：堆叠力量 buff + 多次攻击/X 费攻击 → 单回合伤害爆炸
2. **暴怒/燃尽流**：连续出攻击牌触发暴怒 + 消耗类高伤 → 短期爆发
3. **金属化/护甲流**：金属化 + 屏障 + 反伤 → 用护甲对换伤害

每张卡设计时问自己：这张能进入哪些流派？是否有至少 2 个流派搭配？

## Boss Relic 与卡牌耦合（未来考虑）

战士 Boss 遗物（如「失去能量上限 -1 但获得 4 抽 4」）会强迫某些构筑变形。MVP 暂不实现 Boss 遗物，但卡牌设计时预留兼容性。

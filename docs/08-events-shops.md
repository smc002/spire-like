# 08 事件与商店

## 未知节点解析

进入未知节点（`?` 图标）时按以下概率（可调）决定具体内容：

| 实际内容 | 概率 | 备注 |
|---------|------|------|
| 事件 | 50% | 从未触发过的事件池随机抽 |
| 普通战斗 | 25% | 把这个节点替换为普通战斗 |
| 财宝 | 15% | 直接给金币（小堆） |
| 商店 | 7% | 把这个节点替换为商店 |
| 精英 | 3% | 替换为精英战斗（带警告音） |

这种「未知中再随机」的设计让玩家走每个 ? 都是赌博。

## 事件结构

```gdscript
class_name EventData extends Resource

@export var id: StringName
@export var title: String
@export var image: Texture2D
@export var description: String         # 主文本
@export var options: Array[EventOption]
@export var act: int                    # 限制出现幕
@export var weight: int = 100           # 抽取权重
@export var one_time: bool = true       # 一次跑团内最多触发一次

# 出现条件（可选）：用 GDScript 表达式或自定义脚本
@export var availability_script: GDScript
```

```gdscript
class_name EventOption extends Resource

@export var label: String               # 选项文本
@export var consequence_script: GDScript  # 选了之后的效果
@export var requirement_script: GDScript  # 是否可选（如「需要 50 金」）
@export var result_text: String         # 选后显示的结果文本
```

## MVP 事件原型（5 个）

| 事件 | 选项 |
|------|------|
| 神秘商人 | A: 花 75 金获得稀有遗物 / B: 离开 |
| 自我牺牲 | A: 失去 30%HP 移除一张诅咒/不想要的卡 / B: 不交易 |
| 卡牌轮盘 | A: 获得一张稀有卡，但同时获得一张诅咒 / B: 离开 |
| 古老宝藏 | A: 失去 5%HP 获得 50 金 / B: 离开 |
| 教学之神 | A: 升级一张牌 / B: 改造一张牌 / C: 离开 |

## 商店

### 布局

```
┌────────────────────────────────────────┐
│  卡牌区（5 张随机卡）                  │
│  [卡A 价格] [卡B 价格] [卡C 价格]      │
│             [攻击/技能/能力混合]       │
│                                         │
│  遗物区（3 个，含 1 商店专属）         │
│  [遗物A 价格] [遗物B 价格] [遗物C 价格]│
│                                         │
│  药水区（3 瓶）                         │
│  [药水A] [药水B] [药水C]               │
│                                         │
│  移除服务                              │
│  [移除一张牌 - 当前 75 金]             │
│                                         │
│              [离开]                    │
└────────────────────────────────────────┘
```

### 价格

| 物品 | 基础价 | 备注 |
|------|--------|------|
| 普通卡 | 45-55 | 浮动 ±10% |
| 不普通卡 | 65-75 | |
| 稀有卡 | 135-165 | |
| 诅咒卡 | -- | 不卖 |
| 普通遗物 | 145-165 | |
| 不普通遗物 | 240-260 | |
| 稀有遗物 | 295-315 | |
| 商店专属遗物 | 145-165 | 池子独立 |
| 药水 | 50-100 | 按稀有度 |
| 移除卡 | 75 起，每次 +25 | 跨商店累加 |

数值调到 1 楼平均收益 ~ 1.5 个商店物品。

### 商店随机性

- 每次进入商店 RNG 实例化（按 floor + run seed）
- 价格在范围内随机一次
- 离开商店后再次进入不刷新（避免 save scumming）

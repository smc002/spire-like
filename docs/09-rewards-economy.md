# 09 奖励与经济

## 战斗奖励

战斗胜利后跳到奖励界面，玩家依次领取：

| 奖励 | 出现条件 |
|------|---------|
| 金币 | 必出 |
| 卡牌三选一 | 必出（或选「跳过」） |
| 药水 | 概率出（基础 40%，每次不出下次概率 +10%） |
| 遗物 | 仅精英/Boss |

### 金币

| 战斗类型 | 金币范围 |
|---------|---------|
| 普通战斗（Act 1） | 10-20 |
| 精英战斗（Act 1） | 25-35 |
| Boss（Act 1） | 100 |
| 财宝节点 | 25-35（直接给） |

每个跑团起始金 99（旧梗，但意外好用作初始测试余量）。

### 卡牌三选一

战斗后抽 3 张卡让玩家选 1 张加入卡组。规则：

| 战斗类型 | 卡牌池分布 |
|---------|-----------|
| 普通战斗 | 60% 普通 / 37% 不普通 / 3% 稀有 |
| 精英战斗 | 50% 普通 / 40% 不普通 / 10% 稀有 |
| Boss 战 | 0% 普通 / 0% 不普通 / 100% 稀有 |

约束：
- 3 张卡不重复
- 不出现已有的能力牌（部分能力牌设计为「单实例」）
- 玩家可选「跳过」获得少量金币或药水（设计选项）

### 药水

| 类型 | 效果 |
|------|------|
| 攻击药水 | 立即对一目标造成大伤害 |
| 防御药水 | 立即获得护甲 |
| 状态药水 | 立即获得 X 力量/敏捷（本战斗） |
| 抽牌药水 | 立即抽 3 张 |
| 能量药水 | 立即 +2 能量 |
| 治疗药水 | 立即回 HP |

数量上限 3（基础值，可被某些遗物 +1）。

## 遗物奖励

- 精英战斗：必出 1 个遗物（按精英专属池抽）
- Boss 战斗：三选一稀有遗物
- 宝箱节点：必出 1 个遗物（按当前层稀有度分布）
- 商店：货架上 1-3 个遗物
- 事件：部分事件给遗物

## 经济平衡

Act 1 玩家预期总收入：
- 战斗（约 8 场）：8 × 15 = 120
- 精英（约 2 场）：2 × 30 = 60
- 财宝节点：30
- 起始：99
- **总计：~310 金**

Act 1 玩家预期支出：
- 商店一次：~150（1 件物品 + 1 移除）
- 留作 Act 2 进商店：~150
- 应该够 1 次完整商店购物 + 留一些过 Act

如果实际跑团数据显示玩家长期金币溢出 → 提价 / 多出移除服务 / 商店物品定期刷新（更贵）。

## 升级位置

| 来源 | 数量 |
|------|------|
| 篝火（升级一张卡） | 每篝火 1 张 |
| 事件「教学之神」 | 1 张 |
| 部分稀有遗物 | 战斗开始随机升级 1 张 |

设计前提：玩家在 Act 1 大约升级 3-4 张卡，所以稀有卡升级前后强度差应可控。

## 移除位置

| 来源 | 成本 |
|------|------|
| 商店 | 75 金起，每次 +25 |
| 部分事件 | 失去 HP 换 |
| 「砸坏石头」类事件（如有） | 失去 HP 或物品 |

设计目标：每个 Act 玩家平均移除 1-2 张基础攻击/格挡。

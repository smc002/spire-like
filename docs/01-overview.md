# 01 总览

## 愿景

回合制单人卡牌肉鸽。玩家从塔底出发，逐层爬升，途中组建专属卡组、收集遗物、击败精英与 Boss。每次失败都从头开始，每次跑团的地图、敌人组合、卡牌池都随机生成。

## 核心循环

```
组建初始卡组（固定职业起手） 
  → 进入跑团 
    → 选择下一节点（多分支地图）
    → 战斗/事件/商店/休息
    → 获得奖励（卡牌/金币/可能的遗物或药水）
  → 重复直到 Boss
    → 击败 Boss = 跑团胜利
    → HP 归零 = 跑团失败
```

## 设计支柱

1. **每局唯一**：随机化让玩家在 N 次跑团后仍有新组合。
2. **决策密度**：每回合多张手牌 × 多目标 × 多顺序 = 组合爆炸，但能量上限让选择可控。
3. **明确反馈**：敌人意图永远明示；伤害与状态立即可视化；卡组任意时刻可查看。
4. **死亡有意义**：单次失败结束跑团，但失败教会你下一次怎么构筑。
5. **构筑多样**：同一职业基础牌池能形成 3+ 种有效流派，避免最优解。

## MVP 范围（v1）

| 内容 | 范围 |
|------|------|
| 职业 | 战士（Warrior）一个 |
| 幕数 | Act 1 一幕（约 14-17 个节点） |
| 节点类型 | 战斗、精英、休息、商店、未知、宝箱、Boss |
| 战士卡牌池 | ~35 张（10 基础 + 16 普通 + 9 稀有） |
| 敌人 | 6-10 种（4-5 普通 + 2 精英 + 1-3 Boss） |
| 遗物 | ~12 种（起手 1 + 普通 6 + 不普通 3 + 稀有 2） |
| 事件 | 5 个 |
| 药水 | ~6 种 |

## 非 MVP（先不做）

- 其他职业（沉默者、缺陷、观者）
- Act 2、Act 3、Act 4
- 升华（Ascension）难度
- 每日挑战、自定义跑团
- 成就、统计、解锁系统

## 成功标准（验收）

- 一次完整跑团 25-40 分钟
- 战士基础牌池能跑通至少 3 种构筑（力量流、暴怒/燃尽流、护甲反击流）
- 无显著最优解，多构筑通关率均 > 30%
- 战斗手感：操作流畅、动画 ≤ 0.5s/卡、零卡顿

## 玩家旅程

```
启动游戏 → 主菜单 → [开始新跑团] 
  → 选择职业（MVP 只有战士） 
  → 进入地图（Act 1 第 1 层）
  → 战斗 → 战利品（金币 + 三选一卡牌）
  → 回地图 → 选择下一节点
  → ... 
  → Boss 战
  → 胜利结算 → 返回主菜单
```
